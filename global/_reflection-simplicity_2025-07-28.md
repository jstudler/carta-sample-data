Midway through this project, I notice a pattern: every system I build tends toward complexity. Each new sensor, each additional mapping parameter, each interaction mode adds capability but also adds fragility, calibration burden, and cognitive load for both me and the viewer.

The most successful moments in the Galerie Meridian exhibition were the simplest: one proximity sensor, one slow color shift, one viewer standing still. The complex multi-sensor, multi-parameter configurations I spent weeks developing produced technically impressive but aesthetically diffuse results. There was too much happening — the sculpture became nervous, trying to respond to everything at once.

This is a lesson I have learned before in other contexts but keep needing to relearn. In cooking, the best dishes often have the fewest ingredients. In writing, the clearest prose has the fewest clauses. In responsive sculpture, the best interactions are the ones with the fewest parameters.

The challenge is that restraint requires discipline. It is easier to add than to subtract. Every new capability feels like progress, and removing a feature feels like regression. But the viewer does not see the technical effort — they see only the experience. And a clear, focused experience is almost always more powerful than a complex one.

Going forward, I will adopt a "[one sensor, one response](https://en.wikipedia.org/wiki/KISS_principle)" constraint for initial prototyping: each new piece starts with a single sensor channel and a single light parameter, and complexity is added only if the simple version is demonstrably insufficient.
