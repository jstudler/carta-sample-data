Built the first prototype sensor board today. Using a Luvon Micro 33 as the main controller with the following sensors:

  • BME280 — temperature, humidity, barometric pressure
  • INMP441 — MEMS microphone for ambient sound level
  • VL53L0X — time-of-flight distance sensor for proximity detection
  • VEML7700 — ambient light level

All sensors are communicating over I2C, which keeps the wiring simple. The Micro polls all four sensors at 30 Hz and sends the data over serial to a Crestboard that handles the mapping logic.

First impressions:

  • The BME280 is extremely stable — temperature readings vary by less than 0.1°C over hours.
  • The INMP441 MEMS microphone picks up everything, including the ventilation system. Need significant filtering.
  • The ToF sensor has a useful range of about 1.5 meters, which is roughly the distance at which visitors approach a pedestal-mounted sculpture.
